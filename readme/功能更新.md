# 功能更新记录

## 2026-07-04
- 修复 `GameData/CNSAContracts/Contracts/DFH1_Launch.cfg` 中的解析错误：
  - `deadline` 与 `maxExpiry` 字段需要 `Single`（秒），原写法使用 `Duration()` 导致
    `DataStoreCastException: Cannot cast from Duration to Single`；
  - 改为 `deadline = @/targetDateSeconds - UniversalTime()` 与
    `maxExpiry = @/targetDateSeconds - UniversalTime()`，直接传入剩余秒数。
- 新增 RP-1 科技树补丁 `GameData/CNSAContracts/Patches/CNSA_TechTree.cfg`：
  - 为 DFH-1、CZ1_stage1、CZ1_stage2、CZ1_stage3、CZ1_coupling、CZ1_fairing、
    YF2_Engine、YF2_with_shell、YF3_Engine、YF3_with_shell 共 10 个部件配置
    `@TechRequired`，将其移动到 1950–1960 年代的 RP-1 科技节点；
  - 为上述部件添加 `%RP0conf = true` 与 `%RSSROConfig = true`，使 RP-1 系统正确识别。
- 更新 `GameData/CNSAContracts/Contracts/DFH1_Launch.cfg` 文件头，补充科技树补丁、
  YF-2/YF-3 发动机依赖说明以及 deadline 修正说明。
- 更新 `readme/开发文档.md` 与 `readme/实施文档.md`，补充科技树补丁与 deadline 修正
  相关说明、测试用例。
